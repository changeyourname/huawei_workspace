INTRODUCTION -------------------------------------------------------

This directory contains an example from the OVPsim and CpuManager
User guide. It should be used in conjunction with the user guide but
is briefly described here.

The example shows what information can be extracted from a processor model.

FILES --------------------------------------------------------------
There are three parts to the example
1. An ICM Platform in platform/platform.c
2. Makefile to build 1.

To build the Platform ----------------------------------------------

> make -C platform

RUNNING THE EXAMPLE ------------------------------------------------

> platform/platform.<IMPERAS_ARCH>.exe
